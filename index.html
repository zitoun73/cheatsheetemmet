definition:

Emmet is a plugin for many popular text editors which greatly improves HTML & CSS workflow. It is an essential toolkit for Web Developers. You may have heard Zen Coding before.
It has now changed to Emmet. By instantly expanding simple abbreviations into complex code snippets, Emmet can turn you into a more productive developer.
